Multiway Trie

Also known as a prefix tree, amultiway trieis a tree structure where paths compose words. Edges are labelled with a letter and the node value indicates where the path can terminate; i.e., have we made a word yet?

We can iterate an ascending- or descending-order tree with a pre-order or post-order traversal respectively.
